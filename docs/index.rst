Introduction
------------

harpoon.el is a minimalist port of The Primegean's vim plugin Harpoon to Emacs. It provides quick hotkey switching to selected buffers, allowing you to easily link files to given hotkeys and jump between them efficiently.

Features
--------

- Quick file switching using numeric hotkeys
- Easy file management with a simple configuration file
- Support for up to 4 quick-access files
- Simple and intuitive keybindings
- Works with both regular Emacs and Evil mode

Quick Start
----------

To get started with harpoon.el, you'll need to:

1. Install the package
2. Configure your keybindings
3. Start adding files to your harpoon list

For detailed instructions, see the :doc:`installation` and :doc:`usage` sections.
